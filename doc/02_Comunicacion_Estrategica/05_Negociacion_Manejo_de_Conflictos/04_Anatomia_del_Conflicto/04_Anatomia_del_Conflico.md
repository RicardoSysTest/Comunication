# 04 Anatomia del Conflicto

En esta clase profundizamos en dos temas clave para cualquier líder que negocia: 

1. ¿Cómo se construye un buen acuerdo? 
2. ¿Cómo entender la anatomía de un conflicto? 


## Neogociacion Efectiva
Una negociación efectiva no termina simplemente cuando las partes dicen “sí”, sino cuando se logra un acuerdo que sea sabio, eficiente, duradero y claro.

Un buen acuerdo debe satisfacer los intereses legítimos de las partes, resolver la situación con el menor costo posible y, sobre todo, cuidar la relación para que las personas puedan volver a colaborar en el futuro. Para lograr esto debemos de asegurarnos que los acuerdos deben sean explícitos y específicos: 

* Definir quién hará qué, cuándo, cómo y con qué propósito.

Cuando un acuerdo queda ambiguo o se logra demasiado rápido dejando resentimientos, se convierte en una bomba de tiempo que tarde o temprano reaparecerá como conflicto.

## La anatomía de los conflictos. 
Los conflictos no escalan simplemente por un desacuerdo, sino por el aumento de la carga emocional. Lo que comienza como una diferencia de opinión puede escalar hacia la personalización, el silencio, las suposiciones negativas, la formación de bandos o tribalismo, y en los casos más extremos, la satanización del otro como enemigo.

 **Un conflico implica emociones y percepciones diferentes**

Dado que las emociones intensas bloquean el pensamiento estratégico, el primer paso para resolver un conflicto es bajar la intensidad emocional. Esto puede implicar hacer una pausa, tomar distancia o crear un espacio para recuperar la calma. 

## Grados de Conflicto
A mayor grado mayor mayor gravedad del conflicto y mayor es la dificultad para resolverlo. El conflicto no crece por el desacuerdo si no por el escalamiento emocional, las emociones no reguladas bloqean el pensamiento estrategico

## Problema vs Conflico
Después es importante reconocer explícitamente el problema, separar la intención del impacto y finalmente restaurar la coordinación entre las partes para poder seguir trabajando juntos (un problema es una negociacion fria).

## Pasos para resolver un conflicto
1. Detener la hemorragia: 
   * Desecalada emocional. Nadie escucha cuando esta gritando
2. Nombar al fantasma
   * Reconor explicitamente el problema "Siento tension respecto a X"
3. Separar la intencion del impacto
   * *Tu intencion no fue herir, pero el impacto fue negativo*
4. Restaurar coordinacion
   * El objetivo es volver a trabajar no necesariamente ser amigos.

## Claves para prevenir conflictos. 
* Establecer acuerdos claros por escrito. 
* Mantener retroalimentación frecuente antes de que el malestar crezca y construir una cultura de colaboración donde los errores se vean como problemas técnicos a resolver, no como ataques personales.
